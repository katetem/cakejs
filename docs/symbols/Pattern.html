Class Detail
Pattern(image, repeat)
Pattern is a possibly repeating image that can be used as a strokeStyle or fillStyle. var image = new Image() image.src = 'foo.jpg' var pattern = new Pattern(image, 'no-repeat') var rect = new Rectangle(200, 200, {fill: true, fillStyle: pattern})
Parameters:
image
The image object for the pattern. IMG and CANVAS elements, and Image objects all work.
repeat
The repeat mode of the pattern. One of 'repeat', 'repeat-y', 'repeat-x' and 'no-repeat'. The default is 'repeat'.
